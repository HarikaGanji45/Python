##  Variables & Data Types

Variables store data values.

```python

y = 3.5         # float
name = "Python" # string
is_active = True # boolean
```

##  Input and Output

```python

print("Hello", name)
```


## Operators

* Arithmetic: `+  -  *  /  %`
* Relational: `>  <  >=  <=  ==  !=`
* Logical: `and  or  not`

Example:

```python
x = 10
y = 5
print(x + y)
```


## Type Conversion

```python

```


##  Conditional Statements

```python
x = 10
if x > 5:
    print("Greater than 5")
elif x == 5:
    print("Equal to 5")
else:
    print("Less than 5")
```


##  For Loop

Used to iterate over a sequence.

```python
for i in range(1, 6):
    print(i)
```

Loop with list:

```python
items = ["apple", "banana", "cherry"]
for item in items:
    print(item)
```


##  While Loop

```python
i = 1
while i <= 5:
    print(i)
    i += 1
```


##  Functions

Functions are reusable blocks of code.

### Function Definition

```python
def add(a, b):
    return a + b

result = add(5, 3)
print(result)
```

### Function with No Return

```python
def greet(name):
    print("Hello", name)

greet("Python")
```
